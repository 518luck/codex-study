## 原命令

```bash
ffmpeg -ss 00:01:20 -i input.mp4 -t 6 -vn codex-notify/music/提示音.wav
```

## 整体作用

这条命令的意思是：

从 `input.mp4` 这个视频里，
从 `00:01:20` 这个时间点开始，
截取 `6` 秒钟，
只导出音频，不要视频画面，
最后保存成 `codex-notify/music/提示音.wav`。

## 每一段的意思

### `ffmpeg`

调用 `ffmpeg` 程序来处理音视频文件。

### `-ss 00:01:20`

表示从视频的 `1 分 20 秒` 位置开始截取。

这里的时间格式通常是：

```text
时:分:秒
```

比如：

- `00:00:05` 是第 5 秒
- `00:01:20` 是 1 分 20 秒
- `01:02:03` 是 1 小时 2 分 3 秒

### `-i input.mp4`

指定输入文件。

这里表示输入文件是：

```text
input.mp4
```

也就是你要从这个视频里提取音频。

### `-t 6`

表示截取时长是 `6 秒`。

也就是说：

- 起点是 `00:01:20`
- 长度是 `6 秒`

最终会得到 `00:01:20` 到 `00:01:26` 这段音频。

### `-vn`

表示不要视频流。

`v` 是 video，`n` 可以理解成 no。

加上这个参数后，`ffmpeg` 只保留音频，不输出画面。

### `codex-notify/music/提示音.wav`

这是输出文件路径。

意思是把截出来的音频保存到：

```text
codex-notify/music/提示音.wav
```

其中：

- `codex-notify/music/` 是保存目录
- `提示音.wav` 是输出文件名

后缀写成 `.wav`，表示输出为 `wav` 音频格式。

## 用大白话重新说一遍

这条命令就是：

“从 `input.mp4` 里，找到 `1 分 20 秒` 的位置，往后截 `6 秒`，只要声音，不要画面，最后保存成一个叫 `提示音.wav` 的音频文件。”

## 常见改法

### 如果你想截 10 秒

```bash
ffmpeg -ss 00:01:20 -i input.mp4 -t 10 -vn codex-notify/music/提示音.wav
```

这里只是把：

```text
-t 6
```

改成了：

```text
-t 10
```

### 如果你想从第 30 秒开始截

```bash
ffmpeg -ss 00:00:30 -i input.mp4 -t 6 -vn codex-notify/music/提示音.wav
```

这里只是把开始时间改成了 `00:00:30`。

### 如果你想导出成 mp3

```bash
ffmpeg -ss 00:01:20 -i input.mp4 -t 6 -vn codex-notify/music/提示音.mp3
```

这里只是把输出文件后缀从 `.wav` 改成了 `.mp3`。

通常 `ffmpeg` 会根据输出文件后缀自动选择对应格式。
